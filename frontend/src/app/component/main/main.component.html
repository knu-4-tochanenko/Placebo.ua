Всі ліки з тисячі аптек на одному сайті

Placebo існує для того аби надавати людям інформацію про ліки та їх наявність у аптеках.

Оцініть переваги нашого проекту

Простота функціоналу

На відміну від інших сервісів для пошуку ліків, Placebo дуже простий і зручний у використанні. Наш сервіс відповідає високим технологічним стандартам, однак Ви не побачите тут складних операційних комбінацій. Застосування сучасних інноваційних методик дозволяє зробити процесс пошуку ліків та інформацію про них максимально простим і доступним.

Призначення системи

Даний програмний продукт надає користувачам можливості для пошуку ліків в аптеках міста. Для користувачів збирається інформація про ціни та наявність у різних аптеках на ті лікарські засоби які їх цікавлять.

Інноваційні технології

Важливою перевагою нашого проекту є те, що пошук ліків здійснюється з відкритих джерел. На відміну від інших існуючих сервісів, наш оновлює базу лікарських засобів автоматично за допомогою веб-скрапера, який шукає інформацію на декількох веб-ресурсах. В порівнянні з іншими сервісами, наш має відкрите API (Application Programming Interface), за допомогою якого можна додати пошук за ліками, їх вартістю, наявністю в магазинах, протипокази та інше для інших розробників.

Ефект від вровадження

Ефектом від впровадження системи буде наявність на ринку конкурентно спроможного сервісу для пошуку ліків в Україні, а саме в Києві. Сторонні розробники будуть мати можливість створити програмні продукти розширюючи наявну функціональність завдяки відкритому API
